Here is a list of commonly HTML tags : 1. : The root element of a HTML document. It represents the entire HTML document and contains all other elements. 2. : Contains information about the document, such as the title of the page, meta information, and links to other resources, such as CSS and JavaScript files. 3. : Specifies the title of the document, which is displayed in the browser's title bar or tab. 4. <body>: Contains the main content of the document, such as text, images, and other HTML elements. 5. <header>: Represents a container for introductory content or a set of navigational links. 6. <nav>: Defines a section of a page that contains navigation links. 7. <main>: Specifies the main content of a document, which is unique to that document and excluding content that is repeated across a set of documents, such as site navigation, headers, and footers. 8. <footer>: Represents a container for the footer of a document or section. It typically contains information about the author, copyright information, and links to related documents. 9. <section>: Defines a section of a document, such as a chapter or an introduction. 10. <article>: Specifies independent, self-contained content, such as a blog post or a forum post. 11. <aside>: Represents a section of a page that contains content tangentially related to the main content, such as a sidebar. 12. <h1> to <h6>: Define headings of different levels, with <h1> being the most important and <h6> the least important. 13. <p>: Defines a paragraph of text. 14. <a>: Defines a hyperlink, which links to another page or a specific location on the same page. 15. <img>: Embeds an image in the document. 16. <ul>: Defines an unordered list, which is a list of items with bullet points. 17. <ol>: Defines an ordered list, which is a list of items with numbers or letters. 18. <li>: Defines a list item in an ordered or unordered list. 19. <table>: Defines a table for displaying data in rows and columns. 20. <thead>: Specifies the header content in a table. 21. <tbody>: Specifies the body content in a table. 22. <tr>: Defines a row in a table. 23. <th>: Defines a header cell in a table. 24. <td>: Defines a regular cell in a table. 25. <form>: Defines a form for user input. 26. <input>: Defines a form control where the user can enter data, such as a text field, checkbox, or radio button. 27. <label>: Defines a label for a form control, which provides a description for the control. 28. <select>: Defines a drop-down list for selecting one or multiple options from a list of predefined options. 29. <option>: Defines an option in a drop-down list created with the <select> element. 30. <textarea>: Defines a multi-line text input control. 31. <button>: Defines a clickable button for submitting a form or triggering an action. 32. <style>: Defines a section for CSS styles in a document. 33. <script>: Defines a section for JavaScript code in a document.
